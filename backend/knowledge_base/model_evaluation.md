# 目标检测模型评估

## Precision 与 Recall

Precision 表示预测为缺陷的结果中有多少是真实缺陷；Recall 表示真实缺陷中有多少被模型检出。漏检成本较高时通常更关注 Recall。

## IoU

IoU 是预测框与真实框交集面积除以并集面积，用于衡量定位重合程度。IoU 阈值会影响正负样本判定和 mAP 计算。

## mAP

mAP 是多个类别平均精度的均值。mAP50 使用 IoU=0.5，mAP50-95 则在 0.5 到 0.95 的多个阈值上取平均，对定位质量要求更高。

## 混淆矩阵

混淆矩阵可用于观察类别之间的误判关系。分析时应同时检查背景误检和缺陷漏检，避免只看单一总分。
